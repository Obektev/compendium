Получение последующего члена ряда через предыдущего
$$ C_{n + 1} = C_{n} * T$$
$$
T = \frac{C_{n + 1}}{C_{n}} = \frac{x^2}{(2n + 1)(2n + 2)}
$$
```c++
int main() {
	double x, eps;
	cout << "Введите аргумент и точность: ";
	cin >> x >> eps;
	double ch = 1, y = ch;
	
	for (int n = 0; fabs(ch) > eps; n++) {
		ch *= x*x / ((2 *n + 1) * (2*n + 2));
		y += ch;
	}
}
```
